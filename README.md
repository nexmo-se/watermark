# watermark

This sample shows how to add a watermark to publisher video stream.

Steps:

1. edit app.js and provide your opentok credentials
2. npm install & npm start
3. use ngrok or any other tunnling software to host.
4. open https://yourdomain/index.html in the browser.

This demo makes use of canvas.captureStream so will not work in all browsers. Specificially Safari on Mac and iOS do not support this method.
